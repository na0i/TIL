### 자바의 여러가지 연산자(2)



조건 연산자

- 삼항 연산자: 조건식의 결과에 따라 다른 식이나 결과가 수행됨

- 조건식? 결과 1 : 결과 2

- 조건식이 참이면 결과1을,

- 조건식이 거짓이면 결과2가 선택됨

- ```
  int num = (5>3)? 10:20;
  ```
